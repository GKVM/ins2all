# Set it up

[![Join the chat at https://gitter.im/gkvm/set-it-up](https://badges.gitter.im/gkvm/set-it-up.svg)](https://gitter.im/gkvm/set-it-up?utm_source=badge&utm_medium=badge&utm_campaign=pr-badge&utm_content=badge)

This is a website for installing applications and doing specific functions in a computer.
Installing software is now running a script away.

Steps:
1. Select the applications that has to be installed.
2. Download and run the script

## Currently supported OS
* Linux
* Mac OS
